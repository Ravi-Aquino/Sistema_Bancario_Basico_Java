# 💳 Sistema Bancário em Java

Projeto desenvolvido para praticar Programação Orientada a Objetos (POO) utilizando Java.

O sistema simula operações básicas de um banco diretamente pelo terminal, utilizando armazenamento em memória.

## 🚀 Funcionalidades

- Criar cliente
- Criar conta bancária
- Escolher banco
- Localizar contas
- Depositar
- Sacar
- Transferir entre contas
- Consultar extrato
- Múltiplas contas para o mesmo cliente em bancos diferentes

## 🏦 Bancos disponíveis

- Itaú
- Santander
- Bradesco
- Caixa

## 🛠 Tecnologias

- Java
- Programação Orientada a Objetos
- Collections (ArrayList)
- Separação em camadas (Application, Services, Repository, Entities, View e Util)

## 📁 Estrutura

```
├── application

```
